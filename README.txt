----------
README
----------

The following files are included as part of the project:

1. Report.docx: Final Report with findings and conclusions.
2. main.py: Main Python program that calls the other 3 python files.
3. logistic_model.py: Code that implements a Logistic Model and all necessary charts and statistics for that model.
4. neural_model.py: Code that implements a Neural Network model and all necessary charts and statistics for that model.
5. svm_model.py: Code that implements an SVM model and all necessary charts and statistics for that model.
